# PodcastDeck

PodcastDeck is the Nightshift build 039 submission for 2026-03-24. It is a Vite + React + TypeScript + Tailwind app with a dark mode interface for searching the iTunes podcast catalog, comparing shows, and planning a persistent listening queue.

Live URL: https://obrera.github.io/nightshift-039-podcastdeck/

## Features

- iTunes podcast search with query, country, and result limit controls
- Client-side sorting and explicit-content filtering
- Detailed inspector panel for the selected podcast
- Side-by-side comparison deck for up to three podcasts
- Persistent listening queue with priority, target date, notes, and status
- Queue JSON import/export for backup and transfer

## Development

npm install
npm run build
